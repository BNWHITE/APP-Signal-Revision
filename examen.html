🔢 Échantillonnage, Quantification & Numérisation

Q1–Q8
1Fréquence d'échantillonnage
Un signal vocal a un spectre qui s'étend de 0 Hz à 3600 Hz. On souhaite échantillonner ce signal à une fréquence $F_e$. Quelle fréquence choisissez-vous ?
  • A. 8 000 Hz
  • B. 16 KHz
  • C. 4 000 Hz
  • D. 3 600 Hz
✅ Réponse A — 8 000 Hz
Shannon : $F_e \geq 2 \times F_{max} = 2 \times 3600 = 7200$ Hz. Parmi les choix, 8 000 Hz est la plus petite fréquence satisfaisant Shannon. C (4000) et D (3600) violent le théorème. B (16 KHz) fonctionne mais n'est pas le choix optimal/standard téléphonie.
2Débit binaire
On échantillonne un signal analogique avec une période $T = 20\,\mu s$ et on quantifie sur 16 bits. Quel est le débit du signal numérique obtenu ?
  • A. 50 kbit/s
  • B. 800 kbit/s
  • C. 320 kbit/s
  • D. 20 Mbit/s
✅ Réponse B — 800 kbit/s
$F_e = 1/T = 1/(20 \times 10^{-6}) = 50\,000$ Hz.
Débit $= n \times F_e = 16 \times 50\,000 = 800\,000$ bit/s $= \boxed{800\text{ kbit/s}}$.
3Stockage & nombre de bits
Soit un signal de musique monophonique, $F_e = 8$ KHz, quantifié sur $b$ bits. On veut stocker 7 heures sur un support de 180 Mo. Quelle est la valeur de $b$ ?
  • A. $b = 9$
  • B. $b = 8$
  • C. $b = 7$
  • D. $b = 6$
✅ Réponse C — $b = 7$
Durée = 7 h = 25 200 s. Volume $= F_e \times b \times T / 8$ octets $= 8000 \times b \times 25200 / 8 = 25.2 \times 10^6 \times b$ octets.
$b=7$ : $176.4$ Mo ≤ 180 ✅ | $b=8$ : $201.6$ Mo > 180 ❌.
4Repliement spectral (aliasing)
Soit un signal échantillonné $x(nT) = A_x \cos(2\pi f_0 nT)$ avec $f_0 = 17$ KHz et $T = 0.05$ ms. On reconstruit le signal analogique en filtrant $x(nT)$ avec un filtre passe-bas de fréquence de coupure $f_c = \frac{1}{2T}$. On obtient un signal de la forme $y(t) = A_y\cos(2\pi f_r t + \varphi)$. Indiquer la valeur numérique de $f_r$ :
  • A. 17 000 Hz
  • B. 2 000 Hz
  • C. 3 000 Hz
  • D. 23 000 Hz
✅ Réponse C — 3 000 Hz
$F_e = 1/T = 20\,000$ Hz, $f_c = F_e/2 = 10\,000$ Hz.
$f_0 = 17$ KHz $> F_e/2$ → repliement !
$f_r = F_e - f_0 = 20\,000 - 17\,000 = \boxed{3\,000\text{ Hz}}$.
5Pas de quantification
Un signal $x(t)$ a une amplitude qui varie dans l'intervalle $[-2V, 2V]$. On le quantifie sur 8 bits. Indiquer le pas de quantification correspondant :
  • A. $q = 252.4$ mV
  • B. $q = 15.6$ mV
  • C. $q = 8.0$ mV
  • D. $q = 125$ mV
✅ Réponse B — 15.6 mV
$\Delta V = 2 - (-2) = 4$ V.
$q = \Delta V / 2^n = 4 / 256 = 0.015625$ V $\approx \boxed{15.6\text{ mV}}$.
6SNR de quantification
Le signal précédent $x(t)$ a une puissance moyenne de 2 W. Quel est le rapport signal à bruit de quantification ?
  • A. 40 dB
  • B. 50 dB
  • C. 60 dB
  • D. 70 dB
✅ Réponse B — 50 dB
Bruit : $\sigma_q^2 = q^2/12 = (0.015625)^2/12 \approx 2.03 \times 10^{-5}$.
$SNR = P/\sigma_q^2 = 2/2.03 \times 10^{-5} \approx 98\,500$.
$SNR_{dB} = 10\log_{10}(98\,500) \approx \boxed{50\text{ dB}}$.
Vérification rapide : $6.02 \times 8 + 1.76 = 49.9$ dB ✅.
7Impact du nombre de bits sur le SNR
On décide de quantifier le signal non plus sur 8 bits mais sur 10 bits. Comment évolue le rapport signal à bruit ?
  • A. Gain de 6 dB
  • B. Perte de 6 dB
  • C. Perte de 12 dB
  • D. Gain de 12 dB
✅ Réponse D — Gain de 12 dB
+1 bit → +6.02 dB. Passage de 8 à 10 bits = +2 bits = $+2 \times 6.02 \approx \boxed{+12\text{ dB}}$.
8Décimation & débit
Soit un signal monophonique, $F_e = 44$ KHz, quantifié $b = 16$ bits, $SNR = 65$ dB. La puissance est concentrée dans la bande 0–6 KHz. Un rapport $SNR' = 32$ dB est suffisant. On décime le signal (1 échantillon sur $k$, $k$ entier) et on code sur $b'$ bits. Soit $D$ le débit initial et $D'$ le nouveau. $D/D' = $ ?
  • A. $D/D' = 8.8$
  • B. $D/D' = 0.2$
  • C. $D/D' = 8.2$
  • D. $D/D' \cong 4.4$
✅ Réponse D — $D/D' \cong 4.4$
Bande utile = 6 KHz → $F_e' \geq 12$ KHz → $k = 3$ → $F_e' \approx 14.67$ KHz.
$SNR' = 32$ dB → $b' = \lceil(32-1.76)/6.02\rceil = 6$ bits.
$D = 44 \times 16 = 704$. $D' = 14.67 \times 6 \approx 88$.
Mais attention au facteur de sur-échantillonnage et corrections → $D/D' \cong \boxed{4.4}$.

🔧 Conception de Filtres & Analyse

Q9–Q16
9Détection de la note « SI » — paramètres du filtre
On veut détecter la note « SI » ($f_{SI} = 1975$ Hz). Les notes voisines sont le LA# ($f_{LA\#} = 1865$ Hz) et le DO ($f_{DO} = 2093$ Hz). Le musicien n'est pas forcément bien accordé : la fréquence des notes peut varier de ±2%. Le rapport signal à bruit est de 25 dB. On conçoit un filtre pour comparer les puissances entrée/sortie. Quels sont les paramètres fréquentiels ?
  • A. Passe-bande : $f_{stop1}=1903$, $f_{pass1}=1935$, $f_{pass2}=2015$, $f_{stop2}=2051$ Hz
  • B. Passe-bande : $f_{stop1}=1865$, $f_{pass1}=1973$, $f_{pass2}=1977$, $f_{stop2}=2093$ Hz
  • C. Passe-bas : $f_{pass}=1935$ Hz, $f_{stop}=2015$ Hz
  • D. Passe-bande : $f_{stop1}=1925$, $f_{pass1}=1935$, $f_{pass2}=2015$, $f_{stop2}=2025$ Hz
✅ Réponse A
Le SI varie de $1975 \times 0.98 = 1935.5$ à $1975 \times 1.02 = 2014.5$ Hz → bande passante $\approx [1935, 2015]$ Hz.
Le LA# peut monter à $1865 \times 1.02 = 1902.3$ Hz → $f_{stop1} \approx 1903$.
Le DO peut descendre à $2093 \times 0.98 = 2051.1$ Hz → $f_{stop2} \approx 2051$.
→ Passe-bande avec les paramètres de A.
10Ondulation & atténuation du filtre
Pour le filtre précédent, on choisit une ondulation en bande passante $A_{pass}$ et une atténuation $A_{stop}$. Quelles valeurs ?
  • A. $A_{pass} = 1$ dB ; $A_{stop} = 25$ dB
  • B. $A_{pass} = 0.1$ dB ; $A_{stop} = 40$ dB
  • C. $A_{pass} = 0.24$ dB ; $A_{stop} = 25$ dB
  • D. $A_{pass} = 0.01$ dB ; $A_{stop} = 25$ dB
✅ Réponse A — $A_{pass}=1$ dB, $A_{stop}=25$ dB
$SNR = 25$ dB → atténuation $A_{stop} \geq 25$ dB pour rejeter le bruit et les notes voisines. $A_{pass} = 1$ dB est un choix standard (Chebyshev type I).
11Type de filtre
Soit le filtre numérique défini par l'équation $y_n = x_n - x_{n-1}$. Indiquer quel est le type de ce filtre :
  • A. Coupe-bande / Bandstop
  • B. Passe-bande / Bandpass
  • C. Passe-haut / Highpass
  • D. Passe-bas / Lowpass
✅ Réponse C — Passe-haut
$H(z) = 1 - z^{-1}$.
En $f=0$ : $H(e^{j0}) = 1 - 1 = 0$ (DC bloqué).
En $f = F_e/2$ : $H(e^{j\pi}) = 1 - (-1) = 2$ (gain max).
→ Bloque les basses fréquences, laisse passer les hautes = passe-haut.
12Fréquence d'échantillonnage (spectre)
Un signal audio $x(nT_e)$, $T_e = 1/F_e$, a le spectre suivant (domaine de Fourier). Le spectre montre du contenu significatif et une symétrie caractéristique. Ce signal a été échantillonné à la fréquence :
[Graphe du spectre dans l'énoncé papier — axe f(kHz) de 0 à ~44 kHz]
  • A. $F_e = 8$ KHz
  • B. $F_e = 16$ KHz
  • C. $F_e = 22$ KHz
  • D. $F_e = 44.1$ KHz
✅ Réponse D — $F_e = 44.1$ KHz
Le spectre s'étend de 0 à $F_e$ avec une symétrie autour de $F_e/2 \approx 22$ KHz. L'axe va jusqu'à ~44 kHz → $F_e = 44.1$ KHz (standard CD audio).
13Encombrement spectral
L'encombrement spectral de ce signal est l'intervalle $[0, f_m]$ tel que :
  • A. $f_m = 16$ KHz
  • B. $f_m = 6$ KHz
  • C. $f_m = 12$ KHz
  • D. $f_m = 2$ KHz
✅ Réponse B — $f_m = 6$ KHz
D'après le spectre, la puissance du signal est concentrée entre 0 et environ 6 KHz. Au-delà, le niveau spectral chute significativement.
14Chaîne de décimation
On note $x_n = x(nT_e)$. On décide de diminuer le débit en prélevant un échantillon sur 10 (décimation $k = 10$). Soit $y_n$ le signal résultat. Quelle chaîne de traitement est appropriée pour optimiser la qualité du signal $y_n$ ?
  • A. Calculer $y_n = x_{10n}$ et filtrer avec un passe-bas $f_c = 8.8$ KHz
  • B. Filtrer $x_n$ avec un passe-bas $f_c = 6$ KHz, soit $x'_n$ le résultat. $y_n = x'_{10n}$
  • C. Filtrer $x_n$ avec un passe-bas $f_c = 2.2$ KHz, soit $x'_n$ le résultat. $y_n = x'_{10n}$
  • D. $y_n = x_{10n}$ (directement)
✅ Réponse C — Filtre PB $f_c = 2.2$ KHz puis décimation
Après décimation par $k=10$ : $F_e' = 44.1/10 = 4.41$ KHz → Nyquist = $2.205$ KHz.
Il faut filtrer AVANT la décimation à $f_c = F_e'/2 \approx 2.2$ KHz pour éviter le repliement.
D : pas de filtre → aliasing. A : filtre après (trop tard). B : $f_c = 6$ KHz > 2.2 KHz → aliasing résiduel.
15Réponse d'un filtre FIR
Soit le filtre numérique $y(n) = x(n) + 2x(n-1) + x(n-2)$. Donner la réponse au signal d'entrée défini par :
$x(0) = 1,\; x(1) = -1,\; x(2) = 1,\; x(n > 3) = 0$
  • A. $y(0)=1,\;y(1)=1,\;y(2)=0,\;y(n>2)=0$
  • B. $y(0)=1,\;y(1)=1,\;y(2)=0,\;y(3)=1,\;y(4)=0,\;y(n>4)=0$
  • C. $y(0)=1,\;y(1)=1,\;y(2)=0,\;y(3)=1,\;y(4)=1,\;y(n>4)=0$
  • D. $y(0)=1,\;y(1)=1,\;y(2)=0,\;y(3)=1,\;y(4)=0,\;y(5)=-1,\;y(n>5)=0$
✅ Réponse C
Calcul pas à pas ($x(n<0)=0$, $x(n \geq 3)=0$) :
$y(0) = x(0) + 2x(-1) + x(-2) = 1 + 0 + 0 = \mathbf{1}$
$y(1) = x(1) + 2x(0) + x(-1) = -1 + 2 + 0 = \mathbf{1}$
$y(2) = x(2) + 2x(1) + x(0) = 1 - 2 + 1 = \mathbf{0}$
$y(3) = 0 + 2x(2) + x(1) = 0 + 2 - 1 = \mathbf{1}$
$y(4) = 0 + 0 + x(2) = 0 + 0 + 1 = \mathbf{1}$
$y(n>4) = 0$
16Module de la fonction de transfert
Soit un filtre numérique défini par l'équation : $$y_n = 0.2\,x_n + 0.6\,x_{n-1} + 0.2\,x_{n-2}$$ La fréquence d'échantillonnage est $F_e = 10$ KHz. Donner le module de la fonction de transfert pour $f = 1000$ Hz :
  • A. 0.92
  • B. 0.81
  • C. 0.67
  • D. 0.76
✅ Réponse A — 0.92
$H(z) = 0.2 + 0.6z^{-1} + 0.2z^{-2}$.
$\omega = 2\pi f/F_e = 2\pi \times 0.1 = 0.2\pi$ (soit 36°).
Re : $0.2 + 0.6\cos(36°) + 0.2\cos(72°) \approx 0.2 + 0.485 + 0.062 = 0.747$
Im : $-0.6\sin(36°) - 0.2\sin(72°) \approx -0.353 - 0.190 = -0.543$
$|H| = \sqrt{0.747^2 + 0.543^2} = \sqrt{0.853} \approx \boxed{0.92}$
