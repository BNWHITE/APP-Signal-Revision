🔢 Échantillonnage, Quantification & Numérisation

Q1–Q8
1Fréquence d'échantillonnage — signal audio
Un signal audio a un spectre qui s'étend de 20 Hz à 4400 Hz. On souhaite l'échantillonner sans repliement. Quelle est la fréquence d'échantillonnage minimale parmi les choix suivants ?
  • A. $F_e = 4\,400$ Hz
  • B. $F_e = 11\,025$ Hz
  • C. $F_e = 8\,000$ Hz
  • D. $F_e = 44\,100$ Hz
✅ Réponse B — 11 025 Hz
Shannon : $F_e \geq 2 \times F_{max} = 2 \times 4400 = 8800$ Hz.
A (4400) ❌ viole Shannon. C (8000) ❌ < 8800.
B (11 025) ✅ est la plus petite valeur $\geq 8800$. D marche mais inutilement élevé.
2Débit binaire
Un signal est échantillonné à $F_e = 22\,050$ Hz et quantifié sur 12 bits. Quel est le débit binaire ?
  • A. 22.05 kbit/s
  • B. 176.4 kbit/s
  • C. 264.6 kbit/s
  • D. 529.2 kbit/s
✅ Réponse C — 264.6 kbit/s
$D = F_e \times b = 22\,050 \times 12 = 264\,600$ bit/s $= \boxed{264.6\text{ kbit/s}}$.
3Capacité de stockage
On enregistre un signal stéréo ($\times 2$ canaux) à $F_e = 44\,100$ Hz, quantifié sur 16 bits. Quelle est la taille d'un enregistrement de 3 minutes ?
  • A. 15.9 Mo
  • B. 31.8 Mo
  • C. 63.5 Mo
  • D. 7.9 Mo
✅ Réponse B — 31.8 Mo
$V = 2 \times F_e \times b \times T / 8 = 2 \times 44100 \times 16 \times 180 / 8 = 31\,752\,000$ octets $\approx \boxed{31.8\text{ Mo}}$.
Piège : ne pas oublier le facteur ×2 pour la stéréo !
4Repliement spectral
Soit un signal échantillonné $x(nT) = A\cos(2\pi f_0 nT)$ avec $f_0 = 13$ KHz et $T = 0.1$ ms. On reconstruit le signal analogique avec un filtre passe-bas idéal $f_c = F_e/2$. Quelle est la fréquence $f_r$ du signal reconstruit ?
  • A. 13 000 Hz
  • B. 2 000 Hz
  • C. 7 000 Hz
  • D. 3 000 Hz
✅ Réponse D — 3 000 Hz
$F_e = 1/T = 10\,000$ Hz. $f_c = 5\,000$ Hz.
$f_0 = 13$ KHz $> F_e/2$ → repliement !
$f_r = |f_0 - F_e| = |13\,000 - 10\,000| = 3\,000$ Hz.
On vérifie : $3000 < 5000$ ✅ → passe le filtre.
5Pas de quantification
Un CAN convertit un signal dans l'intervalle $[-5V, +5V]$ sur 10 bits. Quel est le pas de quantification $q$ ?
  • A. $q \approx 9.77$ mV
  • B. $q \approx 4.88$ mV
  • C. $q \approx 19.5$ mV
  • D. $q \approx 39.1$ mV
✅ Réponse A — 9.77 mV
$q = \Delta V / 2^n = (5 - (-5)) / 2^{10} = 10/1024 = 0.009766$ V $\approx \boxed{9.77\text{ mV}}$.
Piège B : utiliser $5V$ au lieu de $10V$ pour la plage.
6SNR de quantification
Un signal sinusoïdal pleine échelle est quantifié sur 12 bits. Quel est le SNR théorique de quantification ?
  • A. 50 dB
  • B. 62 dB
  • C. 74 dB
  • D. 86 dB
✅ Réponse C — 74 dB
Formule pour un signal sinusoïdal pleine échelle :
$SNR = 6.02n + 1.76 = 6.02 \times 12 + 1.76 = 72.24 + 1.76 = \boxed{74\text{ dB}}$.
7Variation du SNR
On passe d'une quantification sur 12 bits à 8 bits. Comment évolue le SNR ?
  • A. Gain de 24 dB
  • B. Perte de 24 dB
  • C. Perte de 12 dB
  • D. Gain de 12 dB
✅ Réponse B — Perte de 24 dB
On perd 4 bits : $\Delta SNR = -4 \times 6.02 \approx \boxed{-24\text{ dB}}$.
Attention au sens ! 12 → 8 = on diminue → perte.
8Sous-échantillonnage
Un signal audio est échantillonné à $F_e = 48$ KHz. Le contenu utile est dans la bande $[0, 4\text{ KHz}]$. On veut sous-échantillonner par un facteur $M$. Quelle est la valeur maximale entière de $M$ ?
  • A. $M = 6$
  • B. $M = 12$
  • C. $M = 5$
  • D. $M = 8$
✅ Réponse A — $M = 6$
Après sous-échantillonnage : $F_e' = F_e/M$. Il faut $F_e'/2 \geq f_{max} = 4$ KHz → $F_e' \geq 8$ KHz.
$M \leq 48/8 = 6$ → $\boxed{M = 6}$.
B (12) donnerait $F_e' = 4$ KHz < $2 \times 4$ = 8 KHz → aliasing.

📊 Analyse Spectrale & Puissance

Q9–Q12
9Puissance d'un signal sinusoïdal
Soit $x(t) = 3\cos(2\pi \times 1000\,t) + 4\sin(2\pi \times 2000\,t)$. Quelle est la puissance moyenne de ce signal ?
  • A. $P = 25$ W
  • B. $P = 7$ W
  • C. $P = 5$ W
  • D. $P = 12.5$ W
✅ Réponse D — 12.5 W
Deux sinusoïdes de fréquences différentes → puissances s'additionnent.
$P = A_1^2/2 + A_2^2/2 = 9/2 + 16/2 = 4.5 + 8 = \boxed{12.5\text{ W}}$.
Piège A : $(3+4)^2/2 = 24.5$ ≠ 25 → on n'additionne pas les amplitudes !
10Conversion dBm
Un signal a une puissance de 0.5 mW. Exprimer cette puissance en dBm :
  • A. $+3$ dBm
  • B. $-3$ dBm
  • C. $-6$ dBm
  • D. $0$ dBm
✅ Réponse B — −3 dBm
$P_{dBm} = 10\log_{10}(P/1\text{mW}) = 10\log_{10}(0.5) = 10 \times (-0.301) = \boxed{-3\text{ dBm}}$.
Rappel : 0 dBm = 1 mW, moitié → −3 dB.
11Résolution fréquentielle de la TFD
On échantillonne un signal à $F_e = 16$ KHz et on calcule une TFD sur $N = 512$ points. Quelle est la résolution fréquentielle $\Delta f$ ?
  • A. $\Delta f = 16$ Hz
  • B. $\Delta f = 64$ Hz
  • C. $\Delta f = 31.25$ Hz
  • D. $\Delta f = 62.5$ Hz
✅ Réponse C — 31.25 Hz
$\Delta f = F_e / N = 16\,000 / 512 = \boxed{31.25\text{ Hz}}$.
12Encombrement spectral
Un signal ECG (électrocardiogramme) a son contenu fréquentiel principalement dans la bande 0.5 Hz à 100 Hz. On choisit $F_e = 500$ Hz. Quel est le rapport de sur-échantillonnage ?
  • A. $R = 2.5$
  • B. $R = 5$
  • C. $R = 500$
  • D. $R = 1$
✅ Réponse A — $R = 2.5$
$F_{Nyquist} = 2 \times f_{max} = 2 \times 100 = 200$ Hz.
$R = F_e / F_{Nyquist} = 500/200 = \boxed{2.5}$.

🔧 Filtrage Numérique

Q13–Q17
13Type de filtre
Soit le filtre numérique : $y_n = x_n + x_{n-1}$. Quel est le type de ce filtre ?
  • A. Passe-haut / Highpass
  • B. Passe-bande / Bandpass
  • C. Coupe-bande / Bandstop
  • D. Passe-bas / Lowpass
✅ Réponse D — Passe-bas
$H(z) = 1 + z^{-1}$.
En $f=0$ : $H(e^{j0}) = 1 + 1 = 2$ (gain max).
En $f = F_e/2$ : $H(e^{j\pi}) = 1 + (-1) = 0$ (zéro).
→ Laisse passer DC, bloque les hautes fréquences = passe-bas.
C'est le complémentaire de $y_n = x_n - x_{n-1}$ (passe-haut).
14Réponse d'un filtre FIR
Soit le filtre $y(n) = x(n) - 2x(n-1) + x(n-2)$. On applique le signal d'entrée :
$x(0)=2,\; x(1)=1,\; x(n)=0$ pour $n \geq 2$ et $n < 0$. Calculer $y(0)$, $y(1)$, $y(2)$, $y(3)$ :
  • A. $y(0)=2,\; y(1)=-3,\; y(2)=0,\; y(3)=1$
  • B. $y(0)=2,\; y(1)=-3,\; y(2)=3,\; y(3)=0$
  • C. $y(0)=2,\; y(1)=1,\; y(2)=-1,\; y(3)=0$
  • D. $y(0)=2,\; y(1)=-3,\; y(2)=0,\; y(3)=-1$
✅ Réponse A
$y(0) = x(0) - 2x(-1) + x(-2) = 2 - 0 + 0 = \mathbf{2}$
$y(1) = x(1) - 2x(0) + x(-1) = 1 - 4 + 0 = \mathbf{-3}$
$y(2) = x(2) - 2x(1) + x(0) = 0 - 2 + 2 = \mathbf{0}$
$y(3) = x(3) - 2x(2) + x(1) = 0 - 0 + 1 = \mathbf{1}$
$y(n \geq 4) = 0$.
15Module de la fonction de transfert
Soit le filtre $y_n = 0.5\,x_n + 0.5\,x_{n-1}$ (moyenne mobile d'ordre 2). $F_e = 8$ KHz. Calculer $|H(f)|$ pour $f = 2000$ Hz :
  • A. $|H| = 1$
  • B. $|H| = 0.5$
  • C. $|H| \approx 0.707$
  • D. $|H| = 0$
✅ Réponse C — $|H| \approx 0.707$
$H(z) = 0.5 + 0.5z^{-1} = 0.5(1 + z^{-1})$.
$\omega = 2\pi f/F_e = 2\pi \times 2000/8000 = \pi/2$.
$H(e^{j\pi/2}) = 0.5(1 + e^{-j\pi/2}) = 0.5(1 + (-j)) = 0.5(1 - j)$.
$|H| = 0.5\sqrt{1^2 + 1^2} = 0.5\sqrt{2} \approx \boxed{0.707}$.
C'est le point à −3 dB (fréquence de coupure !).
16Conception de filtre passe-bande
On veut détecter une fréquence $f_0 = 440$ Hz (le LA) dans un signal audio. Les fréquences voisines à rejeter sont 415 Hz (SOL#) et 466 Hz (SI♭). Les fréquences peuvent varier de ±1%. Quelles sont les fréquences de la bande passante du filtre ?
  • A. $f_{pass1} \approx 436$ Hz, $f_{pass2} \approx 444$ Hz
  • B. $f_{pass1} = 415$ Hz, $f_{pass2} = 466$ Hz
  • C. $f_{pass1} = 430$ Hz, $f_{pass2} = 450$ Hz
  • D. $f_{pass1} = 440$ Hz, $f_{pass2} = 440$ Hz
✅ Réponse A
$f_0 = 440$ Hz ±1% → $[440 \times 0.99, 440 \times 1.01] = [435.6, 444.4]$ Hz.
$f_{pass1} \approx 436$ Hz, $f_{pass2} \approx 444$ Hz.
B inclut les notes voisines (trop large). C arbitraire. D → bande de largeur 0 (impossible).
17Chaîne de décimation
Un signal est échantillonné à $F_e = 32$ KHz avec un contenu utile dans $[0, 2\text{ KHz}]$. On veut décimer par un facteur $M = 4$. Quelle est la bonne chaîne de traitement ?
  • A. Décimer par 4 puis filtrer passe-bas à $f_c = 4$ KHz
  • B. Filtrer passe-bas à $f_c = 8$ KHz puis décimer par 4
  • C. Filtrer passe-bas à $f_c = 4$ KHz puis décimer par 4
  • D. Décimer par 4 directement (pas besoin de filtre)
✅ Réponse C
Après décimation : $F_e' = 32/4 = 8$ KHz → Nyquist = $4$ KHz.
Il faut filtrer AVANT la décimation avec $f_c = F_e'/2 = 4$ KHz pour éviter le repliement.
A : filtre après = trop tard, aliasing déjà présent.
B : $f_c = 8$ KHz > 4 KHz → aliasing résiduel.
D : sans filtre → aliasing total.
